Este proyecto implementa un sistema de gestión de inscripciones basado en una arquitectura de microservicios, utilizando Spring Boot para la lógica de negocio y RabbitMQ para el procesamiento asíncrono de mensajes en formato JSON. La solución permite la recepción de datos estructurados a través de un endpoint REST, los cuales son persistidos automáticamente en una base de datos MySQL, garantizando la integridad y trazabilidad de la información en un entorno de contenedores gestionado por Docker.
